# ReactJS + Hooks +Redux - Building an app like Trello from scratch

## Requirements

- **nodejs <=v14.7.0**
- **npm > = v6.14.7**
- **yarn > = v1.19.1**

clone project and run test.

```

$ get clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ yarn install
$ yarn start
```
